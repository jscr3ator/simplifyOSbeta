# simplifyOS.*cloud*
**The webOS for your self hosted private cloud**


<img width="1080" height="583" alt="simplifyos-the-operating-system-for-your-private-cloud-v0-y8pkyrut9d3g1" src="https://github.com/user-attachments/assets/995feeb4-1fde-4e22-9be4-a99d118a1d4c" />

### Features:
- Modern File Manager
- Text Editor
- Basic Terminal
- Browser to view apps and html files
- App Store

## Setup:

**Manual Installation:**
```bash
git clone https://github.com/jscr3ator/simplifyOS.git
cd simplifyOS
docker compose up --build -d
```
